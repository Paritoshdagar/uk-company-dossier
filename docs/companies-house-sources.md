# Companies House source map

This project uses official Companies House documentation as the authority for API behaviour and public-register caveats.

## Setup and authentication

- [Developer Hub](https://developer.company-information.service.gov.uk/) introduces Companies House developer services.
- [Get started](https://developer.company-information.service.gov.uk/get-started) explains the developer-account path.
- [How to create an application](https://developer.company-information.service.gov.uk/how-to-create-an-application) describes creating an application and API client/key.
- [Authentication](https://developer.company-information.service.gov.uk/authentication) explains API key authentication.
- [Developer guidelines](https://developer.company-information.service.gov.uk/developer-guidelines) includes credential-handling guidance.
- [API testing](https://developer.company-information.service.gov.uk/api-testing) describes testing expectations.

## API references

- [API specifications](https://developer-specs.company-information.service.gov.uk/) is the entry point for current API specs.
- [Companies House Public Data API reference](https://developer-specs.company-information.service.gov.uk/companies-house-public-data-api/reference) covers profile, officers, persons with significant control, charges, insolvency, filing history, and search endpoints used by the dossier and MCP tools.
- [Companies House Document API reference](https://developer-specs.company-information.service.gov.uk/document-api/reference) covers document metadata and content retrieval.
- [Document metadata](https://developer-specs.company-information.service.gov.uk/document-api/reference/document-metadata) is used to resolve filing-document metadata and content links.
- [Fetch a document](https://developer-specs.company-information.service.gov.uk/document-api/reference/document-location/fetch-a-document) is used when retrieving document bytes.

## Data and licensing context

- [Companies House data-products guidance](https://www.gov.uk/guidance/companies-house-data-products) is the starting point for checking data-product terms and attribution expectations.

This repository does not relicense Companies House data. Generated outputs should preserve source attribution and should be checked against the current official register before use.
